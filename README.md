# coralign
Package for the alignment and calibration of coronagraph optics

## Python Package Dependencies
### For general functionality:
1. numpy
2. astropy
3. scipy
4. scikit-image
5. matplotlib
6. pyyaml
7. opencv-python
### For Jupyter notebook examples:
8. jupyterlab
### For unit testing:
9. pytest
### For documentation generation:
10. sphinx
